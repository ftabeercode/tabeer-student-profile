# Tabeer Student Profile

A Flutter-based Student Profile application designed with a modern and responsive user interface.

## Features

- Attractive and responsive UI
- Custom profile picture using CircleAvatar
- Student information display
- University and department details
- Contact information section
- About Me section
- Edit Profile button
- Clean and organized layout
- Cross-platform support (Android, Windows, and Web)

## Technologies Used

- Flutter
- Dart

## Flutter Widgets Used

- MaterialApp
- Scaffold
- AppBar
- Column
- Row
- Container
- CircleAvatar
- Card
- ListTile
- ElevatedButton
- Padding
- SizedBox

## About the Project

This project was developed as part of a Flutter assignment to practice Flutter fundamentals and UI design. The application showcases a student profile with a personalized theme, responsive layout, and an aesthetically pleasing interface.
